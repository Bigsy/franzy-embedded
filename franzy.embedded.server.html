franzy.embedded.server

make-server

(make-server)(make-server broker-config)(make-server broker-config thread-name-prefix)

Creates an Embedded Kafka Server.

A map of broker configuration values may be passed to specify any desired broker parameters, otherwise a default setting is used that will start the broker on your local host with the default ports.

Please prefer passing your own configuration unless the defaults match your desired usage.

make-startable-server

(make-startable-server)(make-startable-server broker-config)

Creates a startable version of an Embedded Kafka Server with slightly less exposed functionality.

This server may be more amenable for command-line usage.
